# 📊 Numerical Methods in Python

This repository contains implementations of fundamental **Numerical Methods** using Python, along with visualizations and method comparisons.

---

## 🚀 Features

* Root Finding Algorithms
* Interpolation Techniques
* Numerical Integration Methods
* Ordinary Differential Equation (ODE) Solvers
* Graphical Visualizations using Matplotlib
* Method Comparisons (Accuracy & Convergence)

---

## 📌 Methods Implemented

### 🔍 Root Finding

* Bisection Method
* Regula Falsi Method
* Secant Method
* Newton-Raphson Method

### 📈 Interpolation

* Lagrange Interpolation
* Newton Forward Interpolation
* Newton Backward Interpolation
* Divided Difference Method

### 📐 Numerical Integration

* Trapezoidal Rule
* Simpson’s 1/3 Rule
* Simpson’s 3/8 Rule

### 📊 Differential Equations

* Euler Method
* Runge-Kutta Method (RK4)

---

## 📊 Visualizations

This project includes graphical representations such as:

* Function plots for root-finding methods
* Iteration convergence points
* RK4 solution curves
* Euler vs RK4 comparison graphs

---

## 🛠️ Tech Stack

* Python
* NumPy
* Matplotlib

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/25sc06005-hub/numerical-methods-python.git
```

2. Navigate into the folder:

```
cd numerical-methods-python
```

3. Install dependencies:

```
pip install numpy matplotlib
```

4. Run any script:

```
python bisection_plot.py
```
